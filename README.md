
# landis & gyr E350 DLMS - COSEM
Read landis &amp; Gyr E350 Electricity Meters for MySensors

# Material

 - Landis & Gyr E350
 - Arduino 2560 pro mini
 - ir sensor
 - nrf24l01
 
 # Protocol and configuration
 
 - DLMS, IEC 62056-21
 - 7 bit, even parity, 1 stop bit, 300 bps
